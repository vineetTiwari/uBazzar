#### This is my rudimentary version of EBay

* Ruby version - 2.2.0

* Rails version - 4.2.0

* Database SQlite

* Other than the basic gems, this application has:

```shell
gem 'bootstrap-sass'
gem 'hirb'
gem 'paperclip'
gem 'paperclip-dropbox'
gem 'rails_12factor'
gem 'pg'
```
* Also required are 'Imagemagick' & 'GhostScript'
